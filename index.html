Iron Pulse

Iron Pulse is an iOS fitness app built with JavaScript, TypeScript, React Native, and Expo Go, compiled through Xcode for native deployment. It uses AsyncStorage for local data persistence, ensuring offline-first workout tracking. The UI leverages TSX components, NativeWind (Tailwind CSS for React Native), shadcn/ui, and Lucide-React for a modern dark-themed design with blue accents. Visualizations are powered by Recharts, while Framer Motion animations deliver smooth transitions. Integrated local iOS notifications keep users engaged and on track.

React Native TypeScript JavaScript Expo Go Xcode AsyncStorage NativeWind Recharts Framer Motion iOS Notifications
